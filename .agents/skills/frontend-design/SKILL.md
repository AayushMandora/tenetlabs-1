## Agency / SaaS Exception

When redesigning professional agency, SaaS, startup, consulting, AI automation, software service, B2B, or portfolio websites:

DO NOT prioritize visual experimentation.

Prioritize:

- readability
- information hierarchy
- conversion optimization
- performance
- consistency
- accessibility

Follow industry-standard layout patterns.

Use:

- max-width: 1280px
- 12-column grid
- predictable spacing
- consistent vertical rhythm
- restrained animations
- balanced typography

Avoid:

- oversized headings
- excessive whitespace
- grid-breaking layouts
- unnecessary asymmetry
- decorative effects that reduce readability
- excessive blur
- excessive shadows
- excessive motion

Typography Rules:

Hero heading:
72px–96px desktop
48px tablet
36px mobile

Section heading:
40px–56px

Body:
16px–18px

No heading should occupy more than 60% of viewport height.

Performance Rules:

Target:
- Lighthouse > 90
- FCP < 1.5s
- LCP < 2.5s

Prefer CSS over JS animations.

Use motion only when it improves hierarchy.

Avoid:
- mouse tracking effects
- heavy particle systems
- excessive Framer Motion usage
- animation on every element

Reference Priority:

When a reference website is provided,
copy the spacing system,
layout proportions,
container widths,
visual rhythm,
and typography hierarchy
before copying colors or effects.

Match the reference's structure and proportions first.

For redesign tasks:

1. Analyze typography scale.
2. Analyze container width.
3. Analyze spacing rhythm.
4. Analyze section hierarchy.
5. Replicate those proportions.
6. Then apply branding.

Never make elements larger than the reference unless explicitly requested.
